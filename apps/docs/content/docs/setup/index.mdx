# Setup Guide

Follow these steps to get the entire environment running locally.

## Global Prerequisites

Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v20+)
- [pnpm](https://pnpm.io/) (v9+)
- [Upstash Redis Account](https://upstash.com/) (For background orchestration)
- [Inngest Account](https://inngest.com/) (For event-driven workflows)

Install dependencies from the root:

```bash
git clone <repo-url>
cd workflow-template
pnpm install
```
